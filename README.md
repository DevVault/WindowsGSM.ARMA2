## Requirements

## Installation
1. Download the [latest](https://github.com/DevVault/WindowsGSM.ARMA2/releases/latest) release
1. Move **ARMA2.cs** and **ARMA2OA.cs** folder to **plugins** folder
1. Click **[RELOAD PLUGINS]** button or restart WindowsGSM
